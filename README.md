How to Layer Sprites 

Start with keyword "show" and then the character you want to show. 
Each character has a default set of mouth, eyes, armL, and armR. 
These can be changed using the descriptors for each character as shown in the sprite layering doc.
Every attribute can only be used once except for the "extras" attribute as described in the doc. 

Attributes that aren't named the way they are in the sprite layering doc:
For armL attributes, include a "l" at the end of the attribute you want.
For armR attributes, include a "r" at the end of the attribute you want.
For any extra attributes, include a "x" at the end of the attribute you want. 

Examples:
show kai1 angry angryx
\newline
show ame1 wink frown foldedl
