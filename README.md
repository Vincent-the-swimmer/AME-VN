How to Layer Sprites 
<br>
Start with keyword "show" and then the character you want to show.<br> 
Each character has a default set of mouth, eyes, armL, and armR.<br>
These can be changed using the descriptors for each character as shown in the sprite layering doc.<br>
Every attribute can only be used once except for the "extras" attribute as described in the doc.<br>
<br>
Attributes that aren't named the way they are in the sprite layering doc:<br>
For armL attributes, include a "l" at the end of the attribute you want.<br>
For armR attributes, include a "r" at the end of the attribute you want.<br>
For any extra attributes, include a "x" at the end of the attribute you want.<br> 
<br>
Examples:<br>
show kai1 angry angryx<br>
show ame1 wink frown foldedl
